# Introduction
该项目为AEMenu项目提供后端支持，更多信息请参照前端项目：
https://github.com/Emma-1024/aemenu

## 技术结构
- Express后端框架
- 文档型数据库MongoDB
- 单元测试使用Jest
- 整合Babel，可使用超前的JavaScript语法

# License
MIT授权的开源项目

# Installation
```
git clone https://github.com/Alex-T-1024/aemenu-backend.git
npm install
```
# Run
前提：MongoDB已安装，DB链接信息在/src/connectDb.js\
DB服务器启动后运行下面命令
```
npm run stable
```
